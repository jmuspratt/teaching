On this page we'll take a different approach to playing the videos. Rather than load them onto the page immediately, we'll hide them and show pre-made images in their place. Then, if one of them is clicked, a jQuery script will hide the image, replace it with its corresponding video, and start playing the video.

Examine the script.js file to see how the jQuery will operate before you get started. Then follow the directions below:

  1. The videos are already on the page. Hide them by using an inline style. Just add style="display:none;" as an attribute to each of the video tags below.
  2. Find the corresponding "poster button" image in the images folder -- this is simply a poster image with a blue play button in the center. For each video, add img tags to the page inside the trigger divs to display the images.
  3. Now wrap those images in a dummy link to make sure the images act like clickable elements
