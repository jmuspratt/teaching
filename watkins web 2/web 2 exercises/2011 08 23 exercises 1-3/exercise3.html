  2. Remember that everything is a box in HTML/CSS. Target the bold text in paragraph2 and add a 1px solid red border. Now add a left margin of 20px and a right margin of -20px (which will cause text to overlap.

Part 2

  1. Below there are three sets of headings, forms, and submit buttons. First, target all formgroups and assign padding, margins, and a light background color to separate them from each other.
  2. The buttons that follow each form are inline elements. To move them below their forms, target them with CSS and assign display:block;
  3. The search form below has a giant submit button (which uses an image file). Make it look better by increasing the size of the form to feel more proportional to the button:
    • Find the HTML for the the search text input.
    • (a) Target the text input element and set a pixel width to expand its size, (b) increase the font size of the input text (you’ll have to type it in yourself when testing), and (c) add padding to push the type off the edges.
    • Finally, separate the form from the submit button by 10px, using margins.
  4. Textareas can take background images. In the images folder, there’s an image called blue-fade.png. Write a rule that assigns it as a background image to the top of textarea1 and make it repeat horizontally.
  5. Remember pseudo-classes? We often use CSS like a:hover {color:blue;} to change styles on mouseover. But there are other pseudo-classes besides hover: form fields (inputs and textareas) can take advantage of the :focus, that kicks in when the user puts their cursor in the field. Write a rule that targets textarea2 and increases the font size when the user clicks into it.

Part 3

  1. The gallery images below are marked up with an unordered list. To get rid of the bullets (added by the browser by default), target the list and use list-style:none;
  2. Remember that every HTML tag is either inline or block by default, but you can override the default with CSS. Target all the list items inside the gallery and set them to display:inline (this is often a good alternative to floating).
  3. Now target the images inside the gallery: give them a background color of #c3ffcb (green) and 20px of padding to reveal the background color.
  4. Every other list item in the gallery have a class of odd assigned to them. Use that class to target the images inside odd images and give them a background color of #c3ebff (blue). Override the left and right padding and increase them to 40px.
