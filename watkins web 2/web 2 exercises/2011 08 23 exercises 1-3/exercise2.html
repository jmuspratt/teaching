Basic Box Model

Make the following changes to the text below using Dreamweaver. You may only edit the CSS.

  1. The images below are 300 px wide and 300 px high. Write a rule that adds a 5px dashed gray border to both of them.
  2. Write a rule that just applies to the first image: add a red background color and 5px of padding to reveal that background.
  3. Add a left margin of 20px to the second image to separate the two images.
  4. The two images and their attributes are now taking up the full width of the main div. Now add 5px of padding to the second image. Since this pushes the second image to the next line, reduce its left margin by the minimum amount necessary to bring it back up to the same line as the first image.
  5. Remember that the box model does notprohibit elements from being larger than their containers — that’s just a default. The paragraph below is nested inside div id="container1". Set a width on #container1 that is much larger than 650px.
  6. Images behave differently: the big Watkins type is a 900px-wide image, nested inside a paragraph and div id="container2". Try setting overflow:hidden; on container2 to see the image get clipped. Now try overflow:scroll;.
