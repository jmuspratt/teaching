Make the following changes to the text below using Dreamweaver. You may only edit the CSS.

  1. Write a rule that sets all paragraphs on the page to be in a gray, sans-serif font. Leave everything else (lists, headings) in Georgia.
  2. Write a rule that makes all paragraphs with a class of instructions green.
  3. Write a rule that puts a red box around all paragraphs with a class of warning. Add some padding to create space inside the box.
  4. Write a rule that sets emphasized text to be red, but only if it appears inside paragraphs that have both the instructions and warning classes as attributes.
  5. Write a rule that adds a dotted gray border above the footer. Use margins and padding to make it improve the spacing.
